🚀 QuarexSMS
Quarex12s tarafından geliştirilmiş, yüksek performanslı SMS gönderim ve yönetim aracı.

🛠 Kurulum ve Çalıştırma
Projenin çalışması için bilgisayarınızda Node.js yüklü olmalıdır.

Dosyaları İndirin:
Projeyi bilgisayarınıza klonlayın veya ZIP olarak indirin.

Bağımlılıkları Yükleyin:
Terminali açıp proje klasörüne gidin ve şu komutu çalıştırın:

Bash
npm install axios chalk dayjs
Başlatın:
Ana dosyayı çalıştırarak sistemi aktif hale getirin:

Bash
node index.js
📋 Özellikler
Hızlı Gönderim: Gelişmiş asenkron döngü sistemi.

Anlık İstatistik: Başarılı/Hatalı işlem takibi.

Renkli Terminal: Chalk ile detaylı işlem logları.
